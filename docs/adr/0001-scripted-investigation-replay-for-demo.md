# Scripted Investigation Replay for Demo

For the hackathon demo, the primary visual path uses a scripted Graph Investigation replay so the Medicine Risk Network, Medicine Risk Graph, Evidence Sources, Graph Change Timeline, and risk-profile updates are deterministic and polished. The scenario should use a real medicine and real Evidence Sources that can be opened live from the demo, while live Neo4j-backed graph data and Graph Intelligence Agent infrastructure can be built in parallel.

**Consequences**:
The scripted path must use the same domain shape as the live system so it remains a credible preview rather than a separate mockup. Any scenario facts that are not backed by verified public sources should be clearly treated as demo data.
