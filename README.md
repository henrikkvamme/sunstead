# Sunstead

Hackathon starter built with TanStack Start, Vite+, Bun, Tailwind CSS, Base UI, Better Auth, Drizzle, and SQLite/libSQL.

## Stack

- TanStack Start + TanStack Router
- Vite+ command surface over Vite, Vitest, Oxlint, and Oxfmt
- Bun for package management and scripts
- Tailwind CSS v4 with Base UI primitives for custom-styled accessible UI
- Zod for server/env validation
- Better Auth for authentication
- Drizzle ORM with SQLite/libSQL

## Commands

```bash
bun install
bun run dev
bun run check
bun run check:fix
bun run build
```

Database commands:

```bash
bun run db:generate
bun run db:migrate
bun run db:studio
```

## Development Environment

This repo includes `devme.toml`.

```bash
devme up -d
devme url web -o
devme doctor
```

On first run, devme writes missing local values to `.env.local`, including `BETTER_AUTH_SECRET`.

## Project Notes

- Oxlint and Oxfmt are configured in `vite.config.ts`, following Vite+ convention.
- `src/routeTree.gen.ts` is generated by TanStack Router and excluded from formatter checks.
- TanStack Start/Router packages are pinned because Start is still moving quickly.
- The default database URL is `file:./local.db`; switch it to a remote libSQL/Turso URL when deploying.
